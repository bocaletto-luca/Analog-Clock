Analog Clock

A PySide6 application displaying a smooth analog clock with hour and minute hands, markers, and customizable colors.

Author: Bocaletto Luca

Source: GitHub Repository

Description

The Analog Clock application renders a real-time analog clock in a graphical window. It features:

  • Current time display with hour and minute hands
  • Minute markers and hour numbers for clear readability
  • Antialiasing for smooth rendering
  • Customizable colors for hands and background
  • Updates every second via a timer

Screenshots

Key Features

  • Hour & minute hands drawn as polygons
  • Anti-aliased QPainter rendering
  • Timer-based updates every second
  • Configurable colors via constants
  • Cross-platform support (Windows, Linux, macOS)

Installation

# Clone the repository
git clone https://github.com/bocaletto-luca/Analog-Clock.git
cd Analog-Clock

# Install PySide6
pip install PySide6

Usage

# Run the clock application
python analog_clock.py

The window will open and show the analog clock updating in real time.

Main Components

  • AnalogClockWindow: Manages the QGuiApplication, rendering, and timer.
  • HOUR_HAND_POLYGON & MINUTE_HAND_POLYGON: Define shapes for the hands.
  • HOUR_COLOR, MINUTE_COLOR, BACKGROUND_COLOR: Color constants.
  • getCurrentTime(): Utility to fetch current local time.

Execution

The script initializes a QGuiApplication, creates an AnalogClockWindow, starts a timer to trigger repaints every second, and enters the main event loop until closed by the user.
